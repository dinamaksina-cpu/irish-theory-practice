Irish Driving Theory Test - update 2.5.1

Changes:
- Search by question number now opens the full ordered question list at that number.
  Example: search 251, then Next goes to 252 and Back goes to 250.
- The last sequential question is saved separately for each profile.
- A Continue card appears on the home screen.
- Statistics now show unique questions completed, progress percentage,
  unique correct answers, unique incorrect answers and accuracy.
- Missing images remain hidden while the app checks available image paths,
  preventing the broken-image icon from flashing.
